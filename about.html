Worlds Without Number is a tabletop role playing game. The rulebook includes an algorithm for generating a point crawl (a framework of nodes and connections which serve as the basis for designing an exploration scenario). This script automates that algorithm.

The source code for this page is available here https://github.com/CamaxtliLopez/mapgen and is released under the GNU General Public License v3.0.
